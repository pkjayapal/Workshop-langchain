Python Environment Setup

1. Create a folder 
2. Open VS Code and then open that folder
3. Open VS Code and go to the Extensions view (Ctrl+Shift+X).
4. Search for and install the Python extension by Microsoft.
5. Open a new terminal (Ctrl+Shift+`). (Cmd Terminal)
6. python -m venv .venv (Creates virtual environment)
7. .\.venv\Scripts\activate (Activates the terminal and you have a PS window)
8. pip install langchain langchain-openai (Install langchain or other libraries)
9. Create a .env file (to store environment variables and keys)
10. Create a hello.py

import os
from dotenv import load_dotenv
from langchain_openai import ChatOpenAI

# Load the API key from your .env file
load_dotenv()

# 1. Initialize the LLM (using a cost-efficient model)
